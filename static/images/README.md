# Logo Files Needed

Add the following files to this directory:

1. **logo.png** - Full Oznog logo (without "Level Up" tagline) for header/hero
   - Transparent background
   - Recommend: 400-600px wide

2. **logo-dark.png** - Same as logo.png (for dark mode, which is default)

3. **favicon.ico** or **favicon.png** - The small winged fist symbol
   - 32x32 or 64x64 pixels

4. **favicon-16x16.png** - 16x16 version
5. **favicon-32x32.png** - 32x32 version
6. **apple-touch-icon.png** - 180x180 version for iOS

Optional:
- **logo-animated.gif** - If you want the glitch/pulse animation in hero
